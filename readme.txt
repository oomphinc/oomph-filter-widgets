== Description ==

Allow back-end users to filter widgets by name for quickly finding the widgets you need.

== Installation ==

1. Upload oomph-filter-widgets to /wp-content/plugins/.
2. Activate plugin through the WordPress Plugins menu.
3. Search widgets on the Widgets administration page by typing in the "Search widget names..." text box, or by pressing Ctrl-Shift-F.
